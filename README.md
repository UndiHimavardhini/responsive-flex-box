# Responsive Layout Using Flexbox

## Description

This project demonstrates a **responsive web layout using CSS Flexbox**.
It includes a navigation bar and a card section that adapt to different screen sizes.

## Tools Used

* HTML5
* CSS3 (Flexbox)
* VS Code

## Features

* Responsive navigation bar
* Flexbox-based card layout
* Mobile-friendly design

## Flexbox Concepts Used

* `display: flex`
* `justify-content`
* `align-items`
* `flex-wrap`
